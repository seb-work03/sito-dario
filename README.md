# sito-dario

Frontend statico della home page di [dariotana.it](https://dariotana.it), estratto dal repo `sito-dario-tana`.

Contiene solo stile e componenti React della home (Next.js + Tailwind CSS, animazioni con Framer Motion). Nessun backend, CMS o API: nessuna dipendenza da Payload CMS, database o form di contatto funzionanti.

## Comandi

- `npm install`
- `npm run dev` — avvia il server di sviluppo
- `npm run build` — build di produzione
